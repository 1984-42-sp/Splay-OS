# Splay OS Product Vision

## What is Splay OS

Splay OS is a free-form visual operating system for decisions, projects, ideas, tasks, notes, files, and meeting memory.

It is not a normal memo app.
It is not only a task manager.
It is a thinking space where ideas can spread, connect, group, and become decisions.

## Core Concept

Splay OS helps the user move from:

- scattered thoughts
- unclear decisions
- loose tasks
- forgotten meeting notes
- disconnected files

toward:

- visible structure
- connected evidence
- grouped ideas
- decision history
- next actions

## Design Philosophy

- Space before structure
- Organic interface
- Decisions have evidence
- Tasks have priority and time
- Workspaces are living spaces
- Empty space has meaning
- Objects should appear only when needed

## Current Use Case

The first real use case is the Takamatsu Cafe Project.

Splay OS is used to organize:

- property research
- business planning
- cafe concept decisions
- finance preparation
- meeting notes
- next tasks
- reference links

## Long-Term Direction

Splay OS should become a personal operating system for thinking and project execution.

Future direction:

- groupable nodes
- multi-select operations
- workspace-level export
- visual reports
- mobile-readable summaries
- 2.5D / spatial interface
- AI-assisted decision support
